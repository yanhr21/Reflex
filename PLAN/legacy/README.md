# Legacy Plan Archive

This directory contains plans that are no longer the active execution protocol.

Archived branches include Cosmos-only world-model plans, low-frequency
executor/scorer plans, contact-action suffix plans, DP handoff plans, and
OpenPI saved-snapshot takeover plans. They may be read as historical evidence
or failure context only.

Do not execute these files as the current plan unless the user explicitly
reactivates a specific branch.
